## Install Dependencies

```
npm ci
```

## Generate types of Prisma

```
cd backend/src/infrastructure/db/ && npx prisma generate && cd -
```

## Run App

```
npm run dev
```

## Run unit test

```
npm run test
```

## Run unit coverage

```
cd frontend && npm run e2e && cd -
```

## Run Cypress (visual)

![Cypress](https://github.com/ankalago/multiplo-test/blob/main/e2e.gif?raw=true)
